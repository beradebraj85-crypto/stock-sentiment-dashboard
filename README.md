## 🛠️ Tech Stack
- **Backend:** Python, FastAPI
- **Frontend:** Plotly Dash
- **Data:** Yahoo Finance API (yfinance)
- **NLP:** TextBlob Sentiment Analysis
- **Charts:** Plotly Candlestick Charts

## ✨ Features
- 📊 Real-time stock prices
- 🧠 AI sentiment analysis on latest news
- 📈 Interactive candlestick charts
- 🔴🟡🟢 Buy/Sell/Hold recommendations
- 📰 Latest news feed for each stock
- 🔍 Multiple stock comparison

## ⚙️ How to Run

### 1. Clone the repository
```bash
git clone https://github.com/beradebraj85-crypto/stock-sentiment-dashboard.git
cd stock-sentiment-dashboard
```

### 2. Create virtual environment
```bash
python -m venv stockenv
stockenv\Scripts\activate
```

### 3. Install dependencies
```bash
pip install -r requirements.txt
```

### 4. Run FastAPI backend
```bash
uvicorn app.main:app --reload
```

### 5. Run Dashboard
```bash
python dashboard.py
```

### 6. Open browser
